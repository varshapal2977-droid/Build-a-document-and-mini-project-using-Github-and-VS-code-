# What is Git?

Git is a version control system that helps developers:

Track changes in code

Save different versions of a project

Undo mistakes

Collaborate without overwriting work

Keep projects safe and organized

# What is GitHub?

GitHub is a cloud-based platform that allows developers to:

Store Git repositories

Share projects publicly or privately

Collaborate with teams

Host websites and documentation

# Git Commands Used
1. git init

Initializes a new Git repository in the project folder.

2. git add

Stages selected files for the next commit.

3. git commit -m "message"

Saves the staged changes with a descriptive message.

4. git push

Uploads commits from local repository to GitHub.

5. git status

Shows the current state of the working directory and staging area.

6. git --version

Displays the installed Git version.
